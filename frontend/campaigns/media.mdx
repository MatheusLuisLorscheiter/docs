## Campanhas com Mídia

Enriqueça suas mensagens com conteúdo visual e documentos.

## Tipos Suportados

<CardGroup cols={4}>
  <Card title="Imagens" icon="image">
    JPG, PNG, GIF
  </Card>
  <Card title="Vídeos" icon="video">
    MP4 (recomendado)
  </Card>
  <Card title="Áudios" icon="microphone">
    MP3, OGG
  </Card>
  <Card title="Documentos" icon="file">
    PDF, DOC, XLS
  </Card>
</CardGroup>

## Como Adicionar

1. Clique em "Adicionar Mídia"
2. Escolha o tipo
3. Faça upload ou cole URL
4. Adicione legenda (opcional)

<Tip>
  Prefira arquivos otimizados (imagens 1280x720, vídeos 720p, áudios 128kbps) para melhor performance de envio.
</Tip>
