## Conexão via QR Code

O QR Code é o método padrão para conectar o WhatsApp ao sistema.

<Note>
  Mantenha o WhatsApp aberto no celular durante todo o processo.
</Note>

## Passo a Passo (pela interface atual)

1. **Abra Configurações**
   - Vá em `Configurações` > seção "Conexões Evolution API"
2. **Gerar QR Code**
   - No cartão da instância com status Desconectado/Conectando, clique em "Gerar QR Code"
3. **Escaneie com o WhatsApp**
   - Siga as instruções abaixo no aplicativo

2. **No WhatsApp Mobile**
   - Abra o WhatsApp
   - Toque em ⋮ (menu)
   - Selecione "Dispositivos conectados"
   - Toque em "Conectar dispositivo"

4. **Escaneie o Código**
   - Aponte a câmera para o QR Code
   - Aguarde a leitura
   - Confirme se solicitado

<Warning>
  O QR Code expira em 60 segundos. Se expirar, gere um novo.
</Warning>

## Solução de Problemas

<Expandable title="QR Code não aparece">
  - Verifique sua conexão
  - Tente outro navegador
  - Desative bloqueadores
</Expandable>

<Expandable title="Não consigo escanear">
  - Limpe a lente da câmera
  - Aumente o brilho da tela
  - Mantenha o celular estável
</Expandable>

<Expandable title="Conexão falha">
  - Verifique internet do celular
  - Atualize o WhatsApp
  - Tente novamente
</Expandable>
