## Acessando o Sistema

O Disparador utiliza um sistema de autenticação seguro para proteger seus dados e campanhas.

<Note>
  Certifique-se de ter suas credenciais de acesso antes de prosseguir. Se ainda não possui, entre em contato com o administrador.
</Note>

## Tela de Login

### Campos de Acesso

<CardGroup cols={2}>
  <Card title="Email" icon="envelope">
    Digite o email cadastrado na empresa
  </Card>
  <Card title="Senha" icon="lock">
    Insira sua senha de acesso
  </Card>
</CardGroup>

### Passo a Passo

1. **Acesse a URL do sistema**
   ```
   https://dispatcher.chavemestragestao.com.br
   ```

2. **Digite suas credenciais**
   - Email: seu-email@empresa.com
   - Senha: sua senha segura

3. **Clique em "Entrar"**
   - O sistema validará suas credenciais
   - Você será redirecionado ao Dashboard

## Recursos da Tela de Login

### 🔐 Segurança
- Conexão criptografada (HTTPS)
- Proteção contra tentativas múltiplas
- Tokens JWT seguros

### 📱 Responsividade
- Interface adaptável para todos dispositivos
- Otimizada para mobile e desktop

### 🌙 Modo Escuro
- Alterna automaticamente com o sistema
- Pode ser configurado manualmente

## Problemas Comuns

<Expandable title="Esqueci minha senha">
  Entre em contato com o administrador do sistema ou suporte técnico:
  - WhatsApp: (48) 98842-0158
  - Email: suporte@chavemestragestao.com.br
</Expandable>

<Expandable title="Email ou senha inválidos">
  Verifique:
  - Se o email está digitado corretamente
  - Se o Caps Lock está desativado
  - Se a senha está correta (atenção a maiúsculas/minúsculas)
</Expandable>

<Expandable title="Sistema não carrega">
  Tente:
  - Limpar o cache do navegador
  - Usar outro navegador
  - Verificar sua conexão com internet
  - Desabilitar extensões do navegador
</Expandable>

## Primeiro Acesso

### Para Novos Usuários

<Warning>
  No primeiro acesso, você precisará:
  1. Receber credenciais do administrador
  2. Fazer login com as credenciais fornecidas
  3. Configurar suas preferências iniciais
</Warning>

### Configurações Recomendadas

Após o primeiro login, recomendamos:

1. **Verificar dados da empresa**
   - Nome da empresa
   - Informações de contato

2. **Configurar notificações**
   - Número para receber relatórios
   - Preferências de notificação

3. **Conectar WhatsApp**
   - Acessar Automações
   - Escanear QR Code

## Sessão e Segurança

### Duração da Sessão
- Sessão ativa por 24 horas
- Renovação automática com uso
- Logout automático por inatividade

### Recomendações de Segurança

<Tip>
  - Use senhas fortes com letras, números e símbolos
  - Não compartilhe suas credenciais
  - Faça logout ao terminar de usar
  - Evite salvar senha em computadores públicos
</Tip>
