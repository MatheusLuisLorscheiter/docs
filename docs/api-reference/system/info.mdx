<Note>
O endpoint de informações fornece dados sobre a versão da aplicação, informações de build, runtime Java e outras configurações do sistema.
</Note>

## Visão Geral

Este endpoint do Spring Boot Actuator fornece informações úteis sobre a aplicação que podem ser usadas para verificar versões, builds e configurações em produção.

## Headers

Nenhum header é obrigatório para este endpoint.

## Uso Prático

### Verificação de Versão

Use este endpoint para verificar qual versão está rodando em produção:

<RequestExample>

```bash Verificar Versão
# Obter apenas a versão
curl -s https://{url-key}/api/actuator/info | jq -r '.app.version'

# Obter informações de build
curl -s https://{url-key}/api/actuator/info | jq '.build'

# Verificar Java runtime
curl -s https://{url-key}/api/actuator/info | jq '.java'
```

```python Script Python
import requests

def get_app_info():
    response = requests.get("https://{url-key}/api/actuator/info")
    if response.status_code == 200:
        info = response.json()
        return {
            'version': info.get('app', {}).get('version'),
            'build_time': info.get('build', {}).get('time'),
            'java_version': info.get('java', {}).get('version')
        }
    return None

info = get_app_info()
if info:
    print(f"Versão: {info['version']}")
    print(f"Build: {info['build_time']}")
    print(f"Java: {info['java_version']}")
```

</RequestExample>
