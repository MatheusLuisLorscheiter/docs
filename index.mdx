# API Disparador

Sistema simplificado e robusto para disparo de mensagens via WhatsApp através da Evolution API.

## 🚀 Características Principais

- **Disparo em Massa**: Envie mensagens para milhares de contatos simultaneamente
- **Agendamento**: Programe campanhas para execução futura
- **Múltiplos Tipos de Mídia**: Suporte a texto, imagens, vídeos, áudios e documentos
- **Variáveis Personalizadas**: Personalize mensagens com dados dos contatos
- **Controle de Status**: Pause, retome ou cancele campanhas em execução
- **Métricas em Tempo Real**: Acompanhe o progresso das campanhas
- **Retry Automático**: Sistema de retry inteligente para mensagens falhadas

## 🏗️ Arquitetura

A API Disparador é construída com tecnologias modernas:

- **Backend**: Spring Boot 3.2.0 com Java 17
- **Banco de Dados**: PostgreSQL para persistência
- **Cache**: Redis para filas e cache
- **WhatsApp**: Integração via Evolution API
- **Documentação**: Swagger/OpenAPI

## 📋 Pré-requisitos

Para usar a API Disparador, você precisa:

1. **Evolution API**: Uma instância da Evolution API configurada
2. **Credenciais**: API Key e URL da sua Evolution API
3. **Instância**: Nome da instância do WhatsApp conectada

## 🔐 Autenticação

A API utiliza um sistema de autenticação simples baseado em tokens:

- Registre sua empresa para obter um access token
- Use o token no header `X-Access-Token` em todas as requisições
- Tokens não expiram (você pode regenerar quando necessário)

## 📱 Tipos de Mensagem Suportados

| Tipo         | Descrição                  | Campos Obrigatórios                |
| ------------ | -------------------------- | ---------------------------------- |
| **TEXT**     | Mensagem de texto simples  | `message`                          |
| **MEDIA**    | Imagem, vídeo ou documento | `message`, `mediaUrl`, `mediaType` |
| **AUDIO**    | Áudio/arquivo de voz       | `message`, `mediaUrl`              |
| **LIST**     | Lista interativa           | `message`                          |
| **BUTTON**   | Botões interativos         | `message`                          |
| **TEMPLATE** | Templates aprovados        | `message`                          |

## 🎯 Fluxo Básico

1. **Registre sua empresa** para obter um access token
2. **Crie uma campanha** com contatos e mensagem
3. **Acompanhe o progresso** através dos endpoints de status
4. **Gerencie campanhas** (pause, retome, cancele)

## 📊 Métricas e Monitoramento

A API fornece métricas detalhadas:

- **Mensagens Enviadas**: Contador de sucessos
- **Mensagens Falhadas**: Contador de falhas com detalhes
- **Status de Campanhas**: Progresso em tempo real
- **Filas**: Status das filas de processamento

## 🔧 Configuração Rápida

```bash
# Clone o repositório
git clone https://github.com/matheusluislorscheiter/api-disparador

# Configure as variáveis de ambiente
export DB_HOST=localhost
export DB_PORT=5432
export DB_NAME=api-disparador
export DB_USER=postgres
export DB_PASSWORD=password
export REDIS_HOST=localhost
export REDIS_PORT=6379

# Execute a aplicação
./mvnw spring-boot:run
```
